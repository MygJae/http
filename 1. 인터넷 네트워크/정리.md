#### 인터넷 통신
- 인터넷 망을 통하여 클라이언트와 서버 통신
#### IP
- 지정한 IP 주소 데이터 전달
- IP 패킷
	- 출발IP
	- 목적IP
	- 데이터
- IP 프로토콜의 한계
	- 비연결성
	- 비신뢰성
	- 프로그램 구분
#### TCP, UDP
- TCP
	- 전송 제어 프로토콜
	- 연결지향
	- 데이터 전달 보증
	- 순서 보장
	- 신뢰할 수 있는 프로토콜
- UDP
	- IP와 거의 같지만 PORT, 체크섬 정도만 추가
#### PORT
- 클라이언트에서 해당 서버에서 프로세스 구분을 위한 것
- 0~1023 잘 알려진 포트, 사용하지 않는 것이 좋음
- IP = 아파트 /  PORT = 몇동 몇호
#### DNS
- DNS 서버에서 도메인 명과 IP등록되어 관리 - 전화번호부 같음
